نظرة عامة على API

توفر منصة Bayena واجهة برمجية RESTful قوية تتيح للمطورين الوصول إلى بيانات البيئة الموحدة والشاملة.

الإصدار الحالي: v1

عنوان الأساس: https://api.bayena.io/v1

المصادقة

جميع طلبات API تتطلب مفتاح API صحيح. يتم نقل المفتاح عبر رأس Authorization.

الحصول على مفتاح API

1. سجل حساباً على لوحة التحكم

2. انتقل إلى قسم API Keys

3. انقر على "إنشاء مفتاح جديد"

استخدام المفتاح

curl -H "Authorization: Bearer YOUR_API_KEY" \ https://api.bayena.io/v1/data/air-quality
أو في JavaScript:
const response = await fetch('https://api.bayena.io/v1/data/air-quality', { headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' } }); const data = await response.json();

الحصول على بيانات نوعية المياه

إرسال بيانات جديدة

POST /data/submit

إرسال بيانات جديدة إلى المنصة

POST /v1/data/submit Content-Type: application/json Authorization: Bearer YOUR_API_KEY { "type": "air-quality", "city": "riyadh", "aqi": 78, "pm25": 35, "timestamp": "2024-03-16T10:00:00Z" }

أمثلة عملية

مثال 1: الحصول على بيانات جودة الهواء

import requests API_KEY = 'your_api_key_here' BASE_URL = 'https://api.bayena.io/v1' headers = { 'Authorization': f'Bearer {API_KEY}', 'Content-Type': 'application/json' } response = requests.get( f'{BASE_URL}/data/air-quality', params={'city': 'riyadh', 'days': 7}, headers=headers ) data = response.json() print(data)

مثال 2: إرسال بيانات جديدة (Python)

import requests from datetime import datetime payload = { "type": "air-quality", "city": "riyadh", "aqi": 78, "pm25": 35, "pm10": 65, "temperature": 28, "humidity": 45, "timestamp": datetime.now().isoformat() } response = requests.post( f'{BASE_URL}/data/submit', json=payload, headers=headers ) print(response.json())

مكتبات وأدوات

Python SDK

pip install bayena-sdk from bayena import BayenaClient client = BayenaClient(api_key='YOUR_API_KEY') data = client.get_air_quality(city='riyadh', days=7) print(data)

JavaScript/Node.js SDK

npm install bayena-sdk const { BayenaClient } = require('bayena-sdk'); const client = new BayenaClient({ apiKey: 'YOUR_API_KEY' }); const data = await client.getAirQuality({ city: 'riyadh', days: 7 }); console.log(data);

معالجة الأخطاء

try { const response = await fetch('https://api.bayena.io/v1/data/air-quality', { headers: { 'Authorization': 'Bearer YOUR_API_KEY' } }); if (!response.ok) { const error = await response.json(); console.error('API Error:', error.message); return; } const data = await response.json(); console.log('Success:', data); } catch (error) { console.error('Network Error:', error); }
